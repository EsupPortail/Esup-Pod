********************************************************************************

    Regenerate Pod theme stylesheets using LESS

********************************************************************************


In this folder you will find:

    • a “settings.less” file containing a list of commented LESS variables you
    can uncomment and edit;

    • a “pod.less” file used to generate the “pod.css” stylesheet found in the
    “assets/css” folder of your theme;

    • an “admin.less” file used to generate the “admin.css” stylesheet found in
    the “assets/css” folder of your theme;

    • an “includes” folder containing files used in “pod.less” and “admin.less”.


You can generate new theme stylesheets (and eventually get them minified) by
using a LESS Compiler (or GUI, like Koala: http://koala-app.com/) on “pod.less”
and “admin.less” after editing “settings.less” and modifying it to suit your
needs.


********************************************************************************

    More info (GUIs, editors, etc.): http://lesscss.org/usage/
